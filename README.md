# Basic Keylogger using Python

This project implements a basic keylogger that records keyboard keystrokes and saves them to a file.

## Description
The program uses the pynput library to listen to keyboard events and logs the keys pressed into a text file.

## Concepts Used
- Keyboard Event Handling
- File Handling
- Logging
- Python pynput Library

## Technologies Used
- Python
- pynput Library

## How to Run
python keylogger.py

Press ESC to stop the keylogger.

This project was developed as part of a cybersecurity internship task.
